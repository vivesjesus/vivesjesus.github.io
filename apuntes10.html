Acceso a ficheros¶

  • Drive¶

Para leer ficheros de tu drive primero se debe montar la carpeta que quieres leer como unidad siguiendo los siguientes pasos:

  1. Importar la librería drive para montar la unidad.
from google.colab import drive
  1. Montar la carpeta a la que queremos acceder.
drive.mount('./drive')

El directorio raíz de Colab es /content.

  • Web¶

# Podemos usar 'wget' desde línea de comando

!wget https://web.ua.es/secciones-ua/images/layout/logo-ua.jpg

  • Fichero local¶

from google.colab import files
files.upload()

Al ejecutarlo, nos aparecerá el típico botón de subida de archivos que abrirá una ventana para seleccionar los archivos que queremos subir.
